**Pill / Tag / Badge / Chip / Card** — the small status & surface vocabulary. Each carries a distinct job; don't substitute one for another.

- **Pill** — identity / presence: `<Pill tone="admin">Admin</Pill>`, `<Pill>Guest</Pill>`, `<Pill tone="online">Online</Pill>`.
- **Tag** — board visibility or a topic tag: `<Tag>private</Tag>`.
- **Badge** — a role beside a name: `<Badge variant="op" />`, `<Badge variant="staff" />`, `<Badge variant="wiki" />`.
- **Chip** — topic status in the inbox / topic header: `<Chip status="solved" />`, `<Chip status="needs" />`, `<Chip status="decision_made" />`. Always shows a word.
- **Card** — the base parchment surface: `<Card>…</Card>` (use `as` to change the tag).

Colour rule: gold = Staff, green = OP/Decision, leaf = Solved, neutral = Pinned/Locked, dashed-faded = Archived.
